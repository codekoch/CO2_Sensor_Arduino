# CO2_Sensor_Arduino
CO2 traffic light warning system based on arduino leonardo

Parts:
- Arduino leonardo
- MH-Z19B (CO2-Sensor)
- 220 Ohm resistor (LEDs)
- 100 Ohm resistor (Buzzer)
- Green, yellow and red LED
- Buzzer
- Switch (Buzzer on/off)
- some wires
- Case
